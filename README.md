
# Home Appliance Management System (Java)

A structured and efficient Java application designed to manage home appliances through a clean, object‑oriented architecture. The system allows users to add, update, view, and search appliance data with persistent storage handled through file‑based I/O.

## ✨ Features
- **Efficient CRUD operations** for managing appliance records.
- **Clean OOP design** with well‑structured classes for appliances, services, and repositories.
- **File‑based persistence**, enabling appliance data to be stored and retrieved across sessions.
- **Robust input validation and error handling** to ensure reliable program behaviour.
- **User‑friendly CLI menu system** for quick and intuitive navigation.
